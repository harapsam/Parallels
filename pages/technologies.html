Curated Datasets

The challenging part of this project compared to the one on Paris real estate is the lack of public data in Morocco. To confront this challenge I compiled two datasets from various sources that contain a wealth of information about people and place.

Dataset 1: maroc_immo.csv

First I built a web scraper to collect data on the web about all of the properties for sale in Morocco; the data included the price, size, location and type of property and a vague description. This created a large dataset of about 6,000 entries. After examining this data, cleaning it up and eliminating false entries, it yielded about 3,500 entries. I also eliminated luxury properties from the dataset because they made it impossible to view the data properly. There were properties selling for 100,000 a square meter! You will not find anything close to that price in Paris. By itself, this does not tell us much. So using a geolocator I was able get the latitude and longitude of each property listing. I also gathered weather data from the closest weather station. In addition, I acquired aquifer type and bore hole flow amount shapefiles from the British Geological Survey and using QGIS I was able to determine the underground water reservoir underneath each property. And finally, again using QGIS, I was able to determine the euclidean distance from the property to the nearest stretch of coastline.

Dataset 2: maroc_city_population.csv

It's not difficult to find population data. However it is a challenge to find detailed population statics for every city and town in Morocco, at least for free. That's where excel is an amazing tool! I was able to take screenshots of extensive tables about population data for 100 cities and towns in Morocco. In excel it is possible to read the data into a table from screenshots. As in the first dataset, I was able cross reference each town with aquifer types and amounts from the British Geological Survey and again using QGIS, calculate the nearest euclidean distance to the sea.
